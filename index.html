markdown语法简介

发表于 2018-04-06

Markdown 目标：易读易写

  • 标题
    MarkDown 支持俩种形式的标题语法 ：
    1. =====H1 —–H2 注意=-可以是任意数量的
    2. #h1 ##h2

  • 列表

    MarkDown 支持有序列表和无序列表 :

    1. 无序列表使用+*-标记
    2. 有序列表使用数字+圆点。
    3. 每个列表项目可以包含多个段落，每个列表项目下的段落都必须缩进4个空格或者是以个制表符

  • 分割线

    你可以在一行中使用— *** ___来做分割线

  • 链接

    这是id链接
    链接可分为行内式 和参考式
    行内式:

    • []+()
      参考式:
    • id: http://www.google.com/ 然后就可以直接在 文中使用如下
      this is id link
  • 强调
    强调

    this is emphasize 利用’**’ 来产生强调的效果或者’_’如下
    this is empthasize

  • 图片引用

    • 一个感叹号！
    • 后边跟着[]替代文字
    • 后边是网址（www.baidu.com)
